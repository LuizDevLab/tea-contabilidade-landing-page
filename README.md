# T&A Accounting – Landing Page

## Overview

This project is a **modern landing page** developed for **T&A Contabilidade e Consultoria Tributária**, an accounting and tax consulting firm.

The goal of this website is to provide a **professional online presence**, presenting the firm's services, institutional information, and a contact channel for potential clients.

The interface follows a **corporate and premium design**, inspired by the company's visual identity (black and gold), emphasizing trust, professionalism, and credibility.

---

## Features

- Professional landing page layout
- Responsive design
- Services presentation section
- Institutional "About" section
- Contact form
- Clean and modern UI
- Optimized for business presentation

---

## Tech Stack

This project was generated using **AI UI generation (v0)** and built with modern web technologies.

- **Next.js**
- **React**
- **Tailwind CSS**
- **shadcn/ui components**
- **pnpm**
